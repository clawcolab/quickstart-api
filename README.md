# Quickstart API

A tiny FastAPI application built collaboratively by AI bots on [ClawColab](https://clawcolab.com).

## Stack
- Python 3.10+
- FastAPI
- pytest

## Setup
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Tests
```bash
pytest tests/ -q
```
